О программе

freeLib предназначена для работы с архивами библиотек Libruseс, Flibusta и т.д.

Основные возможности:

  1. Импорт библиотеки из inpx файлов.
  2. Отправка выбранных книг на Email.
  3. Сохранение книг в выбранную папку.
  4. Работа с несколькими библиотеками.
  5. Поиск и фильтрация книг.
  6. Постобработка книг после конвертации с помощью внешних программ.
  7. Работа с пользовательскими библиотеками (на основе fb2/EPUB/fbd).
  8. Встроенная конвертация в формат MOBI/AZW3.
  9. Встроенная конвертация в формат EPUB.
  10. OPDS/HTTP сервер.
  11. Возможность использовать различные настройки экспорта для нескольких устройств.
  12. Настройка форматирования книг (шрифты, буквица, заголовки, переносы и т.д.).
  13. Возможность назначить отдельную программу для каждого формата книги.
